What is SecurePass?

A free, open-source password generator that creates strong, unique passwords using military-grade cryptography. All processing happens locally‚Äîyour passwords never leave your device.

Key Features:

  • üîê Zero Data Collection - No servers, no tracking.
  • ‚ö° Offline-First - Works without internet.
  • üì± PWA Support - Install as an app for quick access.
  • üî¢ Deterministic Generation - Same input = same password.
  • üíæ Persistent Storage - Domain history saved locally.
  • üöÄ Quick Access - Switch from Safari easily when prompted.
  • üîç Breach Detection - Check if passwords appear in known breaches.

How It Works:

Your master password + domain name are hashed using PBKDF2 with 100,000 iterations to generate a unique password. Domain history is stored locally in your browser.
