🔒 About SecurePass

What is SecurePass?

A free, open-source password generator that creates strong, unique passwords using military-grade cryptography. All processing happens locally—your passwords never leave your device.

Key Features:

  • 🔐 Zero Data Collection - No servers, no tracking.
  • ⚡ Offline-First - Works without internet.
  • 📱 PWA Support - Install as an app.
  • 🔢 Deterministic Generation - Same input = same password.

How It Works:

Your master password + domain name are hashed using PBKDF2 with 100,000 iterations to generate a unique password. No storage needed!
